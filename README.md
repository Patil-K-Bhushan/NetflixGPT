# 🎬 NetflixGPT

NetflixGPT is an AI-powered movie recommendation web application that combines the Netflix browsing experience with OpenAI's GPT-3.5 Turbo API. Users can discover personalized movie recommendations through natural language prompts, browse trending movies, and enjoy a seamless multilingual experience.

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Patil-K-Bhushan/NetflixGPT
```

### 2. Navigate to Project Directory

```bash
cd netflix-gpt
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Create Environment Variables

Create a `.env` file in the root directory:

```env
REACT_APP_OPENAI_API_KEY=your_openai_api_key
REACT_APP_TMDB_API_KEY=your_tmdb_api_key
```

### 5. Start Development Server

```bash
npm start
```
